# University Letter Markup

This is a simple HTML project I made for a course task. In this project, I used HTML to markup a university response letter using different tags.

## What I Used

* HTML
* Basic CSS

## HTML Tags Used

Some tags I used in this project are:

* headings (`h1`, `h2`)
* paragraphs (`p`)
* lists (`ul`, `ol`)
* address (`address`)
* date and time (`time`)
* abbreviation (`abbr`)
* subscript and superscript (`sub`, `sup`)

## About the Project

The purpose of this project was to learn how to structure a letter properly using HTML and make it ready for a website.

I am still learning web development and practicing HTML and CSS.
